## Recent Updates

Sotatek releases updates regularly to enhance your documentation experience. You can find detailed release notes below, organized by version. Each release includes new features, improvements, bug fixes, and occasionally breaking changes.

<Callout kind="info">
  Check the [GitHub releases page](https://github.com/sotatek/docs/releases) for full changelogs, download links, and migration guides.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration for teams, allowing multiple users to edit docs simultaneously.
- Introduced advanced search with filters for tags, authors, and dates.

- Enhanced performance of page rendering by `>30%` for large documentation sites.
- Improved mobile responsiveness with better touch navigation.

## Bug Fixes

- Fixed issue where embedded images failed to load in preview mode.
- Resolved syntax highlighting errors for TypeScript code blocks.

</Update>

## Upgrade Instructions

Follow these steps to update your Sotatek installation. Always back up your data first.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your projects directory.

````bash
tar -czf sotatek-backup-$(date +%Y%m%d).tar.gz /path/to/sotatek/projects
````

  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install sotatek@latest
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add sotatek@latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Migrate Config" icon="settings">

    Update your config file if needed.

<CodeGroup tabs="npm,yarn">
````javascript
// package.json or sotatek.config.js
module.exports = {
  apiKey: `YOUR_API_KEY`,
  theme: {
    primary: '#3B82F6'
  }
};
````
````javascript
// sotatek.config.mjs (new format)
export default {
  apiKey: `YOUR_API_KEY`,
  theme: {
    primary: '#3B82F6'
  }
};
````
</CodeGroup>

  </Step>
  <Step title="Restart Service" icon="refresh-cw">
````bash
sotatek restart
````
  </Step>
</Steps>
